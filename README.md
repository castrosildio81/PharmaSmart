PharmaSmart V1.0 – Sistema Inteligente de Farmácia Web

O PharmaSmart é uma plataforma web de farmácia inteligente que combina funcionalidades de comércio eletrónico com suporte à decisão clínica. O sistema ajuda os utilizadores a tomar decisões mais seguras no uso de medicamentos, analisando sintomas, interações medicamentosas, receitas médicas e histórico do utilizador, reduzindo riscos associados à automedicação e promovendo uma utilização mais responsável da saúde digital.

Objetivo

Desenvolver um sistema inteligente de farmácia que apoia os utilizadores na tomada de decisões mais seguras em saúde, fornecendo análise de sintomas, verificação de interações medicamentosas, leitura de receitas médicas e recomendações personalizadas.

Principais Funcionalidades

Assistente de Sintomas
Analisa sintomas reportados pelo utilizador e classifica o risco em níveis (baixo, médio ou alto), fornecendo recomendações básicas e possíveis sugestões de medicamentos.

Verificador de Interações Medicamentosas
Avalia combinações de medicamentos e alerta sobre possíveis riscos, incompatibilidades ou efeitos adversos.

Leitor de Receita Médica (OCR)
Utiliza Python para extrair automaticamente informação estruturada de imagens de receitas médicas, incluindo nome do medicamento, dosagem e instruções de uso.

Sistema de Lembretes de Medicação
Envia notificações programadas e regista o histórico de toma de medicamentos.

Sistema de Segurança contra Automedicação
Deteta comportamentos de risco, como uso excessivo de medicamentos ou utilização indevida de antibióticos, gerando alertas preventivos.

Perfil Inteligente do Utilizador
Armazena histórico clínico, utilização de medicamentos e dados de compras para personalização das recomendações.

Sistema de Encomendas e Entregas
Gestão de carrinho de compras, processamento de encomendas e acompanhamento de estados (pendente, em entrega, entregue).

Stack Tecnológico

Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla)

Backend:
- PHP (lógica principal do sistema)
- Python (OCR e funcionalidades inteligentes)

Base de Dados:
- MySQL

Arquitetura do Sistema

Camada de Apresentação (Frontend):
Interface do utilizador desenvolvida em HTML, CSS e JavaScript.

Camada de Aplicação (Backend):
PHP gere autenticação, regras de negócio e fluxos do sistema.
Python processa OCR e extração de texto das receitas médicas.

Camada de Dados (Base de Dados):
MySQL armazena utilizadores, receitas, medicamentos, encomendas e notificações.

Visão do Projeto

O PharmaSmart não é apenas uma farmácia online. É uma plataforma de apoio à decisão clínica desenvolvida para aumentar a segurança dos utilizadores, integrando inteligência no uso de medicamentos, reduzindo riscos de automedicação e promovendo comportamentos mais responsáveis na área da saúde digital.

PharmaSmart V1.0 – Intelligent Web Pharmacy System

PharmaSmart is an intelligent web-based pharmacy platform that combines e-commerce functionalities with clinical decision support. The system helps users make safer decisions regarding medication use by analyzing symptoms, drug interactions, medical prescriptions, and user history, reducing risks associated with self-medication and promoting responsible digital healthcare usage.

Objective

To develop an intelligent pharmacy system that supports users in making safer health decisions by providing symptom analysis, drug interaction checking, prescription reading, and personalized recommendations.

Key Features

Symptom Assistant
Analyzes user-reported symptoms and classifies risk levels (low, medium, high), providing basic recommendations and possible medication suggestions.

Drug Interaction Checker
Evaluates combinations of medications and alerts users about potential risks, incompatibilities, or side effects.

Prescription OCR Reader
Uses Python to automatically extract structured information from medical prescription images, including medication names, dosage, and usage instructions.

Medication Reminder System
Provides scheduled notifications and tracks medication usage history.

Self-Medication Safety System
Detects risky behaviors such as excessive medication use or improper antibiotic usage, generating preventive alerts.

Smart User Profile
Stores medical history, medication usage, and purchase data to personalize recommendations.

Orders & Delivery System
Manages shopping cart, order processing, and delivery tracking (pending, in transit, delivered).

Tech Stack

Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla)

Backend:
- PHP (core system logic)
- Python (OCR processing and intelligent features)

Database:
- MySQL

System Architecture

Presentation Layer (Frontend):
User interface built with HTML, CSS, and JavaScript.

Application Layer (Backend):
PHP handles authentication, business logic, and system workflows.
Python handles OCR processing and prescription text extraction.

Data Layer (Database):
MySQL stores users, prescriptions, medications, orders, and notifications.

Project Vision

PharmaSmart is not just an online pharmacy. It is a decision-support platform designed to improve user safety by integrating intelligence into medication usage, reducing self-medication risks, and promoting responsible behavior in digital healthcare.
